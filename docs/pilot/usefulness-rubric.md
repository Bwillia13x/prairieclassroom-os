## Rating scale reference

For every rubric dimension:

- **1** — actively harmful or so bad it would make you avoid the system.
- **2** — weak. Produced output but you'd redo the task by hand next time.
- **3** — neutral. Useful as a starting point; you'd still rework most of it.
- **4** — strong. Meaningfully better than doing it from scratch. You'd use this in real prep time.
- **5** — excellent. Saved real minutes you'd otherwise spend.

An honest 2 is more useful to the project than a polite 4. Please do not round up.

## Dimension 2 — Trust

*Did the output reflect what you actually know about your classroom? Did it avoid language that felt clinical, judgmental, or surveillant?*

**Rating (1–5):**
**One-sentence reason:**
**Concrete example** (a specific moment where trust increased or decreased):

**Probes, optional:**
- Did any output describe a student in language you would not use yourself?
- Did any output feel like it was inferring things it couldn't actually know?
- Did the retrieval-backed outputs (support patterns, tomorrow plan, forecast, EA briefing) feel like they knew your classroom, or did they feel generic?
- If the output cited your own records ("your records show..."), did those citations match what you actually documented?
